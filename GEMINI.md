# Standard Obsidian Library

This library provides standardized utilities for Obsidian plugin development, adhering to the [Google TypeScript Style Guide](https://google.github.io/styleguide/tsguide.html).

It is built upon and functions as an extension to the [Standard TypeScript Library](../standard-ts-lib/GEMINI.md).

## Module Documentation

Below is a list of modules available in this library.

### Decorators
* [decorators.md](src/decorators/decorators.md) - Tracing decorators for `Result` types.

### Filesystem
* [file_util.md](src/filesystem/file_util.md) - Unified Obsidian/Raw filesystem API.

### Schema
* [schema.md](src/schema/schema.md) - Versioned schema management using Zod.
